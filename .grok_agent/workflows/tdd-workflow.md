# TDD Workflow (Multi-Step)

## Steps (Spawn Subagent Per Step)

1. **RED** (Subagent Goal: "Write failing test for [feature]")
2. **GREEN** ("Minimal code to pass test")
3. **REFACTOR** ("Clean code, lint/format")
4. **COMMIT** ("git commit")

**Tools**: make test, tools/tdd.py

Repeat for features.
